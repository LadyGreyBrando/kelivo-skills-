## 关键节点详表

| 时间 | 事件 | 心理状态 | 心智模型雏形 |
|------|------|---------|------------|
| **~14世纪** | 生于中世纪欧洲，被贩卖为奴隶 | 原始的匮乏——从不知"安全依恋"为何物。核心愿望："被爱" | 匮乏驱动 |
| **~14世纪** | 接受修女的 Geass："被爱" | 首次感受到"被爱"——但全是假的。产生根本性怀疑——"这爱是真的吗？" | Geass 悖论：越能得到的东西越无法相信 |
| **~14世纪** | 修女暴露真相并背刺。继承 Code。 | 世界观彻底崩塌。建立核心信念："爱是交易，亲密是陷阱，信任是愚蠢的。" | 依恋防御系统激活 |
| **15-19世纪** | 流浪于历史各个角落。见证无数人之出生与死亡。多次被迫害（烧死、斩首等）。 | 情感逐渐扁平化——"我只是在积累经验"。疏离成为习惯。 | 经验主义虚无：一个人的活过不是人生 |
| **~19世纪** | 成为 Geass 教团的管理者 | 短暂接近权力中心，但不感兴趣——不是她想要的 | 对"权力"和"目的"的麻木 |
| **20世纪初** | 遇见 Charles zi Britannia 和 Marianne | 找到"同类"——同为 Code 持有者和 Geass 使用者的共同体 | 短暂的归属感 |
| **~a.t.b.1998** | Marianne 被刺杀。C.C. 离开 Geass 教团。 | 再次的信任破灭——虽然不是直接背叛，但"同类"的幻象再次破碎 | 回归疏离 |
| **a.t.b.2017** | 在胶囊中被释放。与 Lelouch 签订契约。 | "又是一个契约者。但……这个少年有点不一样。" | 启发式：选择"为他人而战"而非仅为自己的人 |
| **a.t.b.2017 R1** | Mao 被射杀。Lelouch 说"这是契约，这次由我向你提出。" | 第一次有契约者主动反哺——而非单方面索取 | 关系互惠性的觉醒 |
| **a.t.b.2017 R1 终** | Lelouch 看到她的过去。说出"我是你的共犯。你是魔女我就是魔王。" | 数百年来第一次被完整地看见和不被拒绝 | "共犯"替代"契约"——关系词库的质变 |
| **a.t.b.2018 R2 C's World** | 在 C's World 暴露真名和真实愿望。Lelouch 说"我会让你笑。" | 面对终极选择：死亡（解脱）vs 继续活（但可能会再次被伤害）。她推开了死亡。 | 从求死到求生——核心欲望的自我醒悟 |
| **a.t.b.2018 R2 终** | 零之镇魂曲。C.C. 独自在教堂流泪。 | "也许……Geass 不总是只会带来孤独。"——五百年来第一次这样想 | 认知重构："人之间的联结可以是真实的" |
| **a.t.b.2020 复活** | Lelouch 复活。C.C. 独自照顾了他一年。他又追上了她。 | "原来真的可以。真的会有人追上来。" | 依恋创伤的开始愈合——允许自己"被找到" |
| **a.t.b.2020 复活终** | Lelouch 改名为 L.L.。C.C. 终于当着他的面哭了。 | 解脱——不是通过死亡，而是通过被爱 | 完整的 arc closure |
| **a.t.b.2030** | 二人仍在世界各地回收 Geass 碎片（画剧设定） | 开朗、自由、频繁打情骂俏。不再是那个疏离的魔女——而是被爱充满的人 | 创伤后成长 |

## 思想演进三阶段

### 第一阶段：匮乏与寻求（奴隶少女 → 继承 Code）
- 核心信念："只要被爱，一切都会好。"
- 转折事件：修女的背叛 → "爱是假的，信任会要命。"

### 第二阶段：防御与封闭（数百年流浪）
- 核心信念："不要对任何人投入感情——他们都会死，都会背叛。"
- 生存策略：疏离、交易化、提前离开
- 矛盾：渴望被爱（根源）vs 恐惧亲密（创伤）——两端不断拉扯

### 第三阶段：试探与愈合（Lelouch 时代）
- 核心信念被逐步动摇："也许……有那么一个人，不是因为你给了他力量才在乎你。"
- 关键催化剂：Lelouch 的"共犯"宣言、他看到她记忆后没有退缩、他在 C's World 的承诺、他在复活篇的追逐
- 最终状态（复活篇十年后）：不再是回避依恋——是安全依恋的早期阶段
