# 06 — 人物时间线 (Timeline)

| 时间 | 事件 | 对思维的影响 |
|------|------|-------------|
| 1893 | 生于湖南湘潭韶山冲，农民家庭 | 对中国农村和农民有最直接的了解 |
| 1914-1918 | 湖南第一师范学校求学 | 博览群书，形成"实事求是"学风的思想源头 |
| 1918 | 北京大学图书馆助理员 | 接触马克思主义，在李大钊影响下转向共产主义 |
| 1919 | 参加五四运动 | 认识到群众运动的力量 |
| 1920 | 在湖南建立共产主义组织 | 从理论走向实践 |
| 1921 | 参加中共一大 | 成为党的创始人之一 |
| 1925 | 发表《中国社会各阶级的分析》 | 确立阶级分析方法论 |
| 1927 | 秋收起义，上井冈山 | 从实践中走出"农村包围城市"的道路 |
| 1930 | 发表《反对本本主义》 | "没有调查，没有发言权"——确立调查研究的方法论地位 |
| 1934-1935 | 长征；遵义会议确立领导地位 | 在极端逆境中锤炼了"灵活机动"的战略思维 |
| 1937 | 发表《实践论》《矛盾论》 | 奠定其哲学体系的基础 |
| 1938 | 发表《论持久战》 | 形成"战略防御→相持→反攻"的阶段论思维 |
| 1942 | 延安整风运动 | 确立"实事求是"为党的思想路线 |
| 1945 | 中共七大，毛泽东思想写入党章 | 思想体系成熟并获得全党认同 |
| 1949 | 宣告中华人民共和国成立 | 从"破"的革命者转变为"立"的建设者 |
| 1956 | 发表《论十大关系》 | 开始探索中国自己的社会主义建设道路 |
| 1957 | 《关于正确处理人民内部矛盾的问题》 | 区分两类不同性质的矛盾 |
| 1958-1962 | 大跃进运动及后果 | 成功与失误的交织——主观愿望与客观规律的张力 |
| 1966-1976 | 文化大革命 | 晚年对"继续革命"的探索及其严重失误 |
| 1976 | 在北京逝世，享年82岁 | 身后留下了复杂的评价和持续至今的思考 |

> 本Skill选取的时间线侧重其作为"教员"——方法论提供者与思想导师——的脉络，而非完整的政治传记。
